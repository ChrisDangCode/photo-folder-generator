# Photo Folder Generator


Used to generate basic folder structure for the day with support for RAW, pared down Select files, and video files.

```Currently the structure is:

date (year_mo_day) --> 0-RAW
                   |
                   --> 1-jpeg
                   |
                   --> 2-video
                   |
                   --> 3-selects
                   |
                   --> 4-export   --> full-size
                                  |
                                  --> web-size
```
